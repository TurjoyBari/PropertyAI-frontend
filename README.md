# PropertyAI Frontend

Next.js 15 frontend for **PropertyAI** — Real Estate AI Management System.

## Stack

- Next.js 15 (App Router)
- React 19
- TypeScript
- Tailwind CSS

## Getting Started

```bash
cp .env.example .env.local
npm install
npm run dev
```

App: http://localhost:3000

API base URL (backend): set `NEXT_PUBLIC_API_URL` in `.env.local` (default `http://localhost:4000`).
